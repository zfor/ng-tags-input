Ng-Tags-Input

Getting started

Install

  • After downloading all files of ng-tags-input, you have to add the ng-tags-input javascript file to your page

    Make sure it's after the AngularJS script

    <script src="path/to/angular.js"></script>
    <script src="path/to/ng-tags-input.js"></script>

  • And the css

    <link rel="stylesheet" href="path/to/ng-tags-input.css"></script>
<input type="text" class="form-control"/>
<div ng-tags="tags" animate></div>
<div ng-tags="tags"></div>
<div ng-tags="tags" animate 
tag-typeahead="o.value as o.label for o in options"></div>
<div ng-tags="tags" delete-icon></div>
<div ng-tags="tags4" animate tag-typeahead="o for o in options" placeholder="Placeholder"></div>

Hints

Press ctrl+a to select all tags in a tag-input
Press backspace to delete them
